## What it does

CS2 Profile Injector is a Windows-focused utility for handling Counter-Strike 2 profile workflows. It is aimed at users who want a more structured way to edit local profiles, assemble configs, and use preset-based scripting to keep customization tasks organized. The emphasis is on reducing repetitive manual edits while keeping profile data easy to manage.

The tool also offers multilingual UI support, encrypted storage for configuration data, and an assisted generation path that can connect with either the OpenAI API or Claude API. In practice, this makes it useful for managing CS2-related settings like skin changer preferences, crosshair presets, and other config-driven adjustments from a single interface.
